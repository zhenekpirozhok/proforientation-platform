# Proforientation API Documentation

This documentation describes the Proforientation REST API, designed to support
career guidance quizzes, scoring, and recommendations.

The documentation is structured to support both **reference usage** and
**developer onboarding**, following modern API documentation best practices.

## Contents

- OpenAPI reference (`openapi.yaml`)
- Conceptual guides (authentication, pagination, errors, versioning)
- End-to-end onboarding tutorial
- Release notes and changelog

## API Base URL

/api/v1.0

## Target Audience

- Frontend developers
- Mobile developers
- API integrators
- QA engineers

## Design Principles

- RESTful resource modeling
- Stateless authentication (JWT)
- Predictable pagination
- Consistent error format
- Backward-compatible versioning
